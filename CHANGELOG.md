# Changelog

本项目采用版本化变更记录，第一版从 `v0.1.0` 开始。

## v0.1.0

已完成：

- 本地优先 Tauri 桌面应用。
- Vue 3 + TypeScript + Pinia + SQLite 基础架构。
- 5 个示例机房、52 个机柜、208 台设备、12 条告警的 v0.1 示例数据。
- 2D 机柜布局总览、U 位大图、机柜详情、设备详情、搜索定位。
- 资产管理 CRUD、设备校验、Excel 导入预览和重复设备处理。
- AI 模型配置、只读 AI 查询助手、资产位置查询、机柜设备查询、告警查询和 AI 审计记录。
- 告警中心、告警筛选、告警定位和机柜图状态高亮。
- 报表中心，包含设备/机柜/状态/告警/容量/维保基础统计。
- 轻量 3D 机柜视图，支持机柜编号、告警高亮、点击聚焦和领导展示模式。
- 系统设置，包含数据管理、项目 JSON 导入导出、恢复示例数据、主题说明、关于页面。
- 连线管理 v0.1 预留页，保留 `Port`、`Connection`、`DeviceTemplate`、`ChangeRecord` 模型方向。
- Playwright 端到端冒烟测试和手工冒烟清单。

暂不包含：

- 生产级 Prometheus / 卓豪监控适配器。
- 外部 AI Agent 写入 API。
- 多用户权限、角色和完整审计权限模型。
- 复杂 3D 机柜内部模型、线缆拓扑和真实端口占用图。
