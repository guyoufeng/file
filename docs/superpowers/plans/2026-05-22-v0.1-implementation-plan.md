# 泉峰AI数据中心管理平台 v0.1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 交付一个本地可运行的 v0.1 桌面应用，支持 2D 机柜总览、资产管理、Excel 导入、AI 查询、审计日志、基础告警、报表、数据备份和轻量 3D 展示。

**Architecture:** 使用 Tauri 2 作为桌面壳，Vue 3 前端负责页面与可视化，Rust + SQLite 负责本地数据、导入导出和持久化。前端通过 `src/services/backend` 调用 Tauri commands，所有业务模型统一定义在 `src/types/domain.ts`，后续服务化时可以复用同一套 DTO 和边界。

**Tech Stack:** Tauri 2, Vue 3, TypeScript, Vite, Pinia, Vue Router, Naive UI, Konva.js, Three.js, SQLite, sqlx, SheetJS, Vitest, Playwright, OpenAI Compatible style AI Gateway adapters.
